# ELE226
ELE226 burglar alarm project

> [!IMPORTANT]
> This project requires python 3.12 to work due to dependencies of the facial_recognition package

## Instructions for Facial Recognition Demo

1) Navigate into the project directory
2) Run `python3.12 -m venv venv`
3) Run `source venv/bin/activate`
4) Run `pip install -r requirements.txt`
5) Create a directory called `test_images` and place an image of yourself into it
6) From the project directory, run `python src/server/recognise.py`
